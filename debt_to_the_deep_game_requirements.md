# Horror Fishing Game -- Requirements Overview

## Current Confirmed Requirements

### Core Game Concept

-   4-player multiplayer co-op fishing game in 3D
-   Horror element: a shark eats players if they do not catch enough
    fish
-   Daily quota system: catch enough fish to feed the shark

### Multiplayer

-   Up to 4 players
-   Proximity voice chat

### Player Roles

-   Navigator
    -   Drives the boat
    -   Uses map
    -   Tracks weather
    -   Locates jumping fish
-   Fisher
    -   Catches fish from the boat
    -   Uses rods and bait
-   Mechanic
    -   Repairs the boat
    -   Maintains engine and systems
-   Diver
    -   Unlockable through credits
    -   Can dive for treasure and rare fish

### Boat System

-   Upgradeable boat
-   Boat can take damage and require repairs

### Fishing System

-   10 different fishing rods
-   Many fish types
-   Bait system

### Maps

-   5 different maps
-   Different weather conditions
-   Unique map features

### Harbor

-   Safe zone
-   Turn in fish
-   Buy equipment and upgrades

### Items

-   Character customization loot from fishing
-   Buy better bait
-   Buy fishing rods

# Summary

The current design already contains enough features for a solid
prototype or school project:

-   Clear gameplay loop
-   Defined player roles
-   Multiplayer mechanics
-   Upgrade system
-   Multiple maps
-   Item and fishing systems

The optional features can be added later to expand the game after the
core gameplay works.
